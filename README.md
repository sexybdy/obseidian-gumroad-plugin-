# Folio Export

This plugin allows you to easily export one or multiple Markdown notes from your Obsidian vault into a single, beautifully styled HTML document. Perfect for sharing notes with clients, publishing clean web documents, or backing up your data outside of markdown.

## Installation
1. Download the latest release.
2. Extract the files into your application's plugins folder.
3. Restart the application.
## Usage
1. Open the plugin menu from the toolbar.
2. Click on **Export Folio**.
3. Choose your desired output directory and save.
## Features
- Export multiple notes seamlessly
- Maintains clean formatting and CSS styles
- Easy interface built for everyday productivity
